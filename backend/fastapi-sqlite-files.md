# FastAPI + SQLite + 文件哈希存储

## 适用场景

中小体量企业后台：结构化数据 SQLite，大二进制按哈希落盘。

## 模式

| 数据 | 存法 |
|------|------|
| 任务、规则、结果 | SQLite + ORM |
| 上传原件 | SHA-256 去重，分片目录 `{hash[:2]}/{hash}{ext}` |
| 任务快照 | JSON 列冻结规则集 |

## 任务状态

长任务：`进行中 / 待确认 / 完成 / 失败`；进度整数 + statusMessage。

## 时间

DB 存 UTC；API 展示按业务时区（如 Asia/Shanghai）格式化。

## 线程池

| 池 | 用途 |
|----|------|
| 解析池 | 短 CPU（Excel） |
| LLM 池 | HTTP 阻塞调用 |
| 单 worker | 重模型流水线 |

## 反模式

- 大二进制塞 BLOB 表（备份膨胀）
- 执行期读「活」规则表而非快照

## 项目来源

文档审查后端存储，2026 年验证。
